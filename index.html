project 2

개인, 팀 프로젝트(4인, 각자 만든 프로젝트 연동)

오픈뱅킹

2023.06.16 ~ 2023.06.30

Overview

다른 은행 서버 속 내 은행계좌 관리

서로 다른 은행 서버에 존재하는 자신의 계정(같은 이메일)을 하나의 웹 페이지에서 보고 관리하며, 계좌간의 송금과 해당 트랜잭션 유지가 가능한 오픈뱅킹 시스템 입니다.

Preview

api 로그인 및 계좌 송금 시연

순서대로 API 로그인, 송금기록, 페이징 기능, 계좌송금, 송금기록 확인입니다.

Core Features

프로젝트의 핵심 기능
오픈뱅킹 시스템

연결된 서버 속 자신의 이메일로 등록된 계정을 모두 모아볼 수 있는 오픈뱅킹 기능을 구현했습니다.

송금 간 트랜잭션 유지

송금중 데이터베이스 오토커밋을 종료하고, 문제 발생시 rollback, 문제없이 송금 완료시 커밋하는 방식으로 트랜잭션을 구현했습니다.

Open Api 활용

카카오 API를 사용해 카카오 계정 연동을 구현했으며 네이버 뉴스 API를 사용해 금융 관련 뉴스를 출력하는 기능을 구현했습니다.

Full Features

프로젝트의 전체 기능들

- 로그인, 회원가입, 개인정보 수정, 회원 탈퇴
- 카카오 api를 통한 로그인
- 계좌 개설 및 계좌 정보 관리
- 오픈뱅킹
- 계좌 송금
- 네이버 api를 활용한 금융 뉴스 모아보기
- 송금액 및 여타 금액 실시간 원화로 보조 표기 (ex: 1234억 567만 80원)
- 거래내역 생성 및 확인
- 게시판 기능
- 무한 대댓글 기능
- 거래내역, 게시판 페이징 기능
- 유저 정보 확인 (관리자)
- 게시판 관리 (관리자)

project 3

팀 프로젝트(4인)

모션인식 헬스케어 (프로젝트 진행중)

Overview

모션인식을 통해 얻은 데이터를 웹서버에서 관리

파이썬 앱의 MediaPipe를 통한 모션인식을 통해 사용자의 운동을 보조하고, 그렇게 만들어진 데이터를 웹서버로 전송해 활용하는 기능을 가진 시스템입니다.

2023.07.11 ~ 진행중

Core Features

프로젝트의 핵심 기능
모션인식

MediaPipe와 카메라를 통해 모션 인식을 도입했습니다. 모션 인식을 통해 사용자의 자세가 올바른지 판단할 수 있습니다.

JSON을 통한 어플과 웹 서버 사이의 통신

모션인식 파이썬 앱을 통해 만들어진 데이터를 서버에 저장하고, 서버에서 해당 데이터를 활용해 각종 서비스를 제공할 수 있습니다.

Full Features

프로젝트의 전체 기능들

- 로그인, 회원가입, 회원정보 수정, 회원탈퇴
● 모션인식을 통한 운동 메뉴 선택
● 모션인식 보조 운동
● JSON을 통한 어플 - 웹서버간 통신

● 표시가 있는 기능은 제가 구현한 기능입니다.

- 차후 더 많은 기능을 추가할 예정입니다.
